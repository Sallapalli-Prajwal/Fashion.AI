## Project Overview

Fashion.AI is a Single Page Application (SPA) that:
- Connects to Google Photos via OAuth
- Analyzes outfit photos using Google Vision API
- Provides style recommendations using Google Gemini API
- Stores analysis results in Google Firestore
- Displays analytics and insights to users

**Tech Stack:**
- Frontend: React (Vite)
- Backend: Express.js / Node.js
- Cloud: Google Cloud Run
- Database: Google Firestore
- APIs: Google Vision API, Google Gemini API, Google Photos API
- Analytics: Google Analytics (GA4)

# GitHub Wiki Setup Instructions

## How to Use These Wiki Pages

1. **Go to your GitHub repository**: https://github.com/Sallapalli-Prajwal/Fashion.AI

2. **Enable Wiki** (if not already enabled):
   - Go to repository Settings
   - Scroll to "Features" section
   - Enable "Wikis"

3. **Create Wiki Pages**:
   - Click "Wiki" tab in your repository
   - Click "New Page" for each page
   - Copy content from the corresponding `.md` file in `github-wiki/` directory
   - Paste into the wiki page editor
   - Save the page

4. **Add Screenshots**:
   - For each page, look for "ðŸ“¸ SCREENSHOT REQUIRED HERE" markers
   - Take the required screenshots
   - Upload images to the wiki (use the image upload button)
   - Insert images where marked

7. **Code** - Link to repository

## Quick Start Guide

1. **Enable Wiki** in repository settings
2. **Create each page** using the content from `github-wiki/*.md` files
3. **Take screenshots** as you work through the application
4. **Upload screenshots** to wiki pages where marked
5. **Create PDFs** for Presentation, Analytics Report, and Proposal
6. **Upload PDFs** to repository or link them in wiki
7. **Upload video** to YouTube (unlisted) and link it

---

## Tips

- **Screenshots**: Use a tool like Snipping Tool (Windows) or Screenshot (Mac)
- **Annotations**: Use image editing to add arrows/notes if needed
- **Organization**: Keep screenshots organized in a folder before uploading
- **Naming**: Name screenshots descriptively (e.g., "dashboard-after-login.png")
- **Quality**: Ensure screenshots are clear and readable
- **Privacy**: Blur or hide any sensitive information (emails, tokens, etc.)

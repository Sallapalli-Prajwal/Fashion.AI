## How to Take Screenshots

### Windows
- **Snipping Tool**: Windows + Shift + S
- **Print Screen**: PrtScn key
- **Snipping Tool App**: Search "Snipping Tool" in Start menu

### Mac
- **Screenshot Tool**: Cmd + Shift + 4
- **Full Screen**: Cmd + Shift + 3
- **Window**: Cmd + Shift + 4, then Space

### Browser DevTools
- **F12** or **Right-click â†’ Inspect**
- **Network Tab**: For API calls
- **Console Tab**: For logged output
- **Application Tab**: For cookies/storage

---

## Screenshot Tips

1. **Use high resolution** - Ensure text is readable
2. **Hide sensitive data** - Blur emails, tokens, personal info
3. **Add annotations** - Use arrows/boxes to highlight important parts
4. **Be consistent** - Use same browser/window size
5. **Name files clearly** - e.g., "dashboard-after-login.png"
6. **Organize in folders** - Group by wiki page
7. **Check file size** - Compress if too large (>5MB)
